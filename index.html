Milestones & Company Value

Define the five stages and the company value at each. Company value, type %, and member % all carry five milestone values; later milestones inherit earlier ones until you override them.

Distribution by Type

Each type's share of the whole, per milestone. The starred (✱) type auto-balances so the four sum to 100.00% in every milestone.

Distribution by Member

Names are editable. Each type splits evenly by default; override any member and the starred (✱) member absorbs the difference. Values are absolute (share of the whole), per milestone.
